Introduction to Logarithmic and Exponential Functions

A review of the fundamental properties and calculus rules for logarithmic and exponential functions.

Firstly, it is well known that Exponential and Logarithmic Functions are inverses of each other. That is, they satisfy the fact that:

$$f(f^{-1}(x))=f^{-1}(f(x))=x$$

That means,

$$\log_{a} (a^{x}) = a^{\log_{a}(x)}=x$$

as long as the base $a$ remains constant throughout.

In addition, we also know how to change logarithms from one base to the other. For example, if you want to change a logarithm of $x$ in base $a$ to base $b$, we use:

$$\log_{a}x=\frac{\log_{b}x}{\log_{b}a}$$

In MAT121, we looked at the differentiation and integration of the basic exponential and logarithmic functions. Here, we discussed that:

  • The derivative of an exponential function $y=e^x$ is $e^x$. When the base is $e$, it is called Natural Exponential Function.
  • For any other base $a$, we found the derivative of the function $y=a^x$ to be $a^x \ln a$.
  • For logarithms, we discussed that the natural logarithmic function of x is present as $\ln x$ as a short form of $\log_{e} x$.
  • We also discussed that the derivative of the function $y=\ln x$ is $\frac{1}{x}$. And for any other general base $a$ we first change it to base $e$ and then differentiate which yielded the derivative of $y=\log_{a} x$ as $\frac{1}{x \ln a}$.
  • On Integration, we found the reverse processes of the above derivatives. And for most questions we used integration by substitution to solve them.

SUMMARY

Derivatives:

$$\begin{aligned} & \frac{d}{dx}\left(e^{u(x)}\right)=e^{u(x)}\times \frac{du}{dx}\\ &\frac{d}{dx}\left( a^{u(x)}\right) = a^{u(x)}\ln a\times\frac{du}{dx} \\ & \frac{d}{dx}\left( \ln u(x)\right) = \frac{1}{u(x)}\times \frac{du}{dx}\\ &\frac{d}{dx}\left( \log_{a}u(x)\right)=\frac{1}{u(x)\ln a}\times\frac{du}{dx} \end{aligned}$$

Integrals:

$$\begin{aligned} &\int e^{u(x)} \, du=e^u+c\\ &\int a^{u(x)} \, du=\frac{a^{u(x)}}{\ln a}+c\\ &\int \frac{1}{u(x)}\, du = \ln |u(x)|+c \end{aligned}$$

Logarithmic Differentiation

The technique of logarithmic differentiation simplifies complex functions by using logarithms to transform products, quotients, and exponents into simpler expressions.

The other thing that we looked at in MAT121 is the Quotient Rule which dealt with the differentiation of rational functions. However, some rational functions are so complicated such that using the quotient rule seems almost impossible.

For example, consider the function below:

\[y=\frac{x^5}{(1-10x)\sqrt{x^2+2}}\]

The above function can be solved using the quotient rule but the process will be very complicated to follow. Therefore we will be using Logarithmic Differentiation.

In this process of differentiation, we will be introducing natural logarithms on both sides and solve the function implicitly (Check Implicit differentiation too).

Given \(y=\frac{x^5}{(1-10x)\sqrt{x^2+2}}\), introduce natural logarithms both sides.

\[\ln (y)=\ln \left(\frac{x^5}{(1-10x)\sqrt{x^2+2}} \right)\]

We then apply the rules of indices and logarithms where we know that:

  • If expressions are multiplying, then their logs are adding
  • If expressions are dividing, then their logs are subtracting
  • A power is a product of a log
\[\ln y = \ln x^5 - \ln\left((1-10x)\sqrt{x^2+2} \right) =\ln x^5 - \left[ \ln(1-10x)+\ln (x^2+2)^{\frac{1}{2}} \right]\]
\[\ln y = \ln x^5 -\ln(1-10x)-\frac{1}{2}\ln (x^2+2)\]

After this, we will differentiate implicitly; each term on its own using the derivative formula where in simplified terms, for \(y=\ln u \text{, } y'=\frac{u'}{u}\).

Let's now use implicit differentiation to differentiate \(\ln y = \ln x^5 -\ln(1-10x)-\ln \sqrt{x^2 +2}\) term by term as follows:

\[\begin{aligned} &\frac{d}{dx}(\ln y)=\frac{1}{y}\times\frac{dy}{dx}=\frac{y'}{y} \text{ since }\frac{dy}{dx}=y'\\ &\frac{d}{dx}(\ln x^5)=\frac{5x^4}{x^5}=\frac{5}{x}\\ &\frac{d}{dx}(\ln(1-10x))=\frac{-10}{1-10x}\\ &\frac{d}{dx}\left( \frac{1}{2}\ln (x^2+2)\right) =\frac{1}{2}\times \frac{2x}{x^2+2}=\frac{x}{x^2+2} \end{aligned}\]

From here, we just need to combine everything and simplify if possible.

For example, consider the function below; \[ y=\frac{x^5}{(1-10x)\sqrt{x^2+2}} \] The above function can be solved using the quotient rule but the process will be very complicated to follow. Therefore we will be using the Logarithimic Differentiation

The other thing that we looked at in MAT121 is the Quotient Rule which dealt with the differentiation of rational functions. However, some rational functions are so complicated such that using the quotient seems almost impossible.

Therefore, the derivative of \(y=\frac{x^5}{(1-10x)\sqrt{x^2+2}}\) is:

\[\frac{y'}{y}=\frac{5}{x}-\left( \frac{-10}{1-10x}\right) - \frac{x}{x^2 + 2}=\frac{5}{x}+\frac{10}{1-10x} - \frac{x}{x^2 + 2}\]

If we multiply by \(y\) on both sides:

\[y'=\left( \frac{5}{x}+ \frac{10}{1-10x}- \frac{x}{x^2 + 2}\right) \times y\]

But \(y=\frac{x^5}{(1-10x)\sqrt{x^2+2}}\), if we substitute we get the final derivative of \(y\) as:

\[\frac{dy}{dx}=\left( \frac{5}{x}+ \frac{10}{1-10x}- \frac{x}{x^2 + 2}\right) \left( \frac{x^5}{(1-10x)\sqrt{x^2+2}}\right) \]

We can see here that it's slightly simpler to do the differentiation using logarithms than it would be if we had used the product and quotient rules. In addition, the answers are also simplified which improves our understanding of the process.

The same approach is also used to differentiate expressions in which functions are raised to other functions. Thus:

\[\text{If } y=f(x)^{g(x)} \text{, then, } \ln y = \ln \left( f(x)^{g(x)}\right)=g(x)\ln f(x)\]

Then, we apply implicit differentiation and use the product rule on the right hand side to finish the differentiation process.

Example 1: Differentiate \(y=x^x\)

Solution

  • Firstly, we can happily see that both power rule \(y'=nx^{n-1}\) and the exponential differentiation \(y'=a^x \ln a\) won't work. So, let's introduce the logarithms.
\[\ln y = \ln x^x = x\ln x\]
  • Here, we will differentiate \(\ln y\) implicitly and \(x\ln x\) using the product rule.
\[\frac{1}{y}\cdot \frac{dy}{dx}=1\cdot\ln x + x\cdot\left( \frac{1}{x}\right) =\ln x + 1\]
  • Multiplying by \(y\) both sides yields:
$$\frac{dy}{dx}=(\ln x + 1)x^x$$

Example 2: Given that \(y=(1-3x)^{\cos(x)}\), find \(\frac{dy}{dx}\)

Solution

  • Introduce logarithms:
\[\ln y = \ln \left[(1-3x)^{\cos(x)}\right] = \cos(x)\ln (1-3x)\]
  • Differentiate:
\[\frac{1}{y}\cdot\frac{dy}{dx}=-\sin (x)\cdot\ln(1-3x)+\cos(x)\cdot\frac{-3}{(1-3x)}=-\sin(x)\ln(1-3x)-\frac{3\cos(x)}{(1-3x)}\]
$$\therefore \frac{dy}{dx}=-\left[\sin(x)\ln(1-3x)+\frac{3\cos(x)}{(1-3x)} \right]\left[(1-3x)^{\cos(x)} \right]$$

Before we look at more examples, let's revise some of the basic differentiation formulas to avoid confusion.

\(\frac{d}{dx}(c)=0\) Constant Differentiation
\(\frac{d}{dx}(x^n)=nx^{n-1}\) Power Rule
\(\frac{d}{dx}(a^x)=a^x\ln a\) Derivative of an Exponential Function
\(\frac{d}{dx}(x^x)=x^x(1+\ln x)\) Logarithmic Differentiation

It's very useful to keep an eye on the theorems we use to avoid costly mistakes since a lot of things look similar but are mathematically different.

Example 3: Differentiate \(f(x)=(5-3x^2)^7\sqrt{6x^2+8x-12}\)

Solution

  • Introduce logarithms both sides
    • $$\ln f(x)=\ln\left[(5-3x^2)^7\sqrt{6x^2+8x-12} \right]$$ $$=\ln(5-3x^2)^7+\ln(6x^2+8x-12)^{\frac{1}{2}}$$
  • Simplify this using rules of Indices and logarithms
    • $$\ln f(x)=7\ln(5-3x^2)+\frac{1}{2}\ln(6x^2+8x-12)$$
  • Now, we have to differentiate both side implicitly.
    • $$\frac{f'(x)}{f(x)}=7\left(\frac{-6x}{5-3x^2} \right) +\frac{1}{2}\left( \frac{12x+8}{6x^2+8x-12} \right)$$
  • Simplify
    • $$f'(x)=f(x)\left( \frac{-42x}{5-3x^2}+\frac{6x+4}{6x^2+8x-12}\right) $$ $$\therefore f'(x)=(5-3x^2)^7\sqrt{6x^2+8x-12}\left[ \frac{-42x}{5-3x^2}+\frac{6x+4}{6x^2+8x-12}\right]$$

Example 4: Differentiate \(y=(2x-e^{8x})^{\sin 2x}\)

Solution

$$\ln y = \ln \left(2x-e^{8x} \right)^{\sin 2x}=\sin 2x \ln(2x-e^{8x})$$ $$\frac{y'}{y} = (2\cos 2x)\cdot\ln(2x-e^{8x})+\sin 2x\cdot\left(\frac{2-8e^{8x}}{2x-e^{8x}} \right)$$ $$y'=\left(2\cos 2x\ln(2x-e^{8x})+\frac{\sin 2x(2-8e^{8x})}{2x-e^{8x}} \right) \times y$$ $$\therefore y'=\left[2\cos 2x\ln(2x-e^{8x})+\frac{\sin 2x(2-8e^{8x})}{2x-e^{8x}} \right]\left[ (2x-e^{8x})^{\sin 2x}\right]$$

Logarithmic Integration

Understanding how to use logarithms in integration problems.

So far, we have seen that the derivative of \(\ln x\) is \(\frac{1}{x}\). And let's say we want to integrate \(\frac{1}{x}\) using the power rule: \(\int x^n \, dx = \frac{x^{n+1}}{n+1}+c\).

Firstly we will write \(\frac{1}{x}\) as \(x^{-1}\) using the rules of indices. If we apply the power rule here:

\[\int x^{-1}\, dx = \frac{x^{-1+1}}{-1+1}=\frac{x^0}{0}=\frac{1}{0}\]

which is undefined.

Now, knowing the fact that the integral is an antiderivative of a given function, we can happily conclude that \(\ln x\) is an antiderivative of $\frac{1}{x}$ on the interval \((-\infty,0)\cup(0,\infty)\) that doesn't contain \(0\). Therefore:

\[\int \frac{1}{u} \, du = \ln |u|+C\]

Example 1: Evaluate \(\int\limits_1^e \frac{1}{x} \, dx\)

Solution

  • We will firstly find the indefinite integral then handle the limits later. Thus:
\[\int \frac{1}{x} \, dx = \ln |x| + C\]
  • Bringing the limits in:
\[\int\limits_1^e \frac{1}{x}\, dx = [\ln x]_1^e=\ln e - \ln 1=1-0=1\]

From rules of logarithms, we know that \(\ln e = \log_{e}e = 1\text{ and } \ln 1 = 0\).

Example 2: Evaluate \(\int \frac{3x^2}{x^3 +5}\, dx\)

Solution

  • We will be using Integration by Substitution. The idea is to eliminate a certain part of the expression by substituting a derivative of another part of the given expression. Here, let \(u=x^3+5\), after differentiating \(\frac{du}{dx}=3x^2\). Making \(dx\) a subject of the formula, we get \(dx=\frac{du}{3x^2}\). Hence we will substitute \(u \text{ and } dx\):
\[\int \frac{3x^2}{u}\cdot\frac{du}{3x^2}=\int \frac{1}{u}du=\ln |u| +C\]
  • And substituting \(u\) back yields:
\[\int \frac{3x^2}{x^3 +5}\, dx = \ln |x^3+5|+C\]

Example 3: Evaluate \(\int \tan x \, dx\)

Solution

  • Here, we will first have to apply the trig identity, \(\tan x = \frac{\sin x}{\cos x}\).
\[\int \tan x \, dx = \int \frac{\sin x}{\cos x}\, dx\]

Let \(u=\cos x\) and \(du=-\sin x \, dx\)

  • Thus:
\[\int \frac{\sin x}{\cos x} \, dx =-\int \frac{du}{u} = -\ln |u| + C\]
  • Substituting \(u\) back:
\[\int \tan x \, dx = - \ln |\cos x | + C\]

Example 4: Integrate \(\int \frac{dx}{x\ln x}\)

Solution

  • Firstly, we need to make a choice of \(u\) that will eliminate \(dx\) as shown:
\[\text{Let } u = \ln x \text{; Then }du =\frac{1}{x}\, dx\]
  • Integrating the resulting expression yields the following:
\[\int \frac{dx}{x\ln x} = \int \frac{du}{u} = \ln |u|+C\]
  • Substituting \(u\) back:
\[\int \frac{dx}{x\ln x}= \ln |\ln x|+C\]

Example 5: Evaluate \(\int \frac{\sin 3\theta}{1+\cos 3\theta}\, d\theta\)

Solution

\[u=1+\cos 3\theta \quad \quad du = -\sin 3\theta \, d\theta\]
\[\int \frac{\sin 3\theta}{1+\cos 3\theta}\, d\theta = -\int \frac{du}{u}=-\ln |u|+C\]
\[\therefore \int \frac{\sin 3\theta}{1+\cos 3\theta}\, d\theta = -\ln |1+\cos 3\theta| +C\]

Example 6: Evaluate \(\int \cot x \, dx\)

Solution

\[\cot x = \frac{\cos x}{\sin x} \quad \quad u= \sin x \text{, and } du = \cos x \, dx\]
\[\int \cot x \, dx = \int \frac{\cos x}{\sin x} \, dx = \int \frac{du}{u} = \ln |\sin x| + C\]

Inverse Trigonometric Functions

Understanding the derivatives of inverse trigonometric functions and their applications.

Mathematically, the six basic trigonometric functions do not have inverses because their graphs repeat periodically and do not pass the horizontal line test.

However, inverses of trigonometric functions are widely used in finding angles associated with specific ratios. These inverses usually give angles in the range \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\). For example:

\[sin x = \frac{1}{2}; \quad x = \sin^{-1}\left( \frac{1}{2} \right) = 30^\circ\]

When differentiating inverse trigonometric functions, careful attention must be taken as they behave differently from regular inverse functions.

To derive the derivative of an inverse function, we use the identity:

\[g'(x) = \frac{1}{f'(g(x))}\]

This identity holds if \(f(x)\) and \(g(x)\) are inverse functions: \(f(g(x)) = x\) and \(g(f(x)) = x\).

Note: \(\sin^{-1}x = \arcsin x\), and similar for other functions.

Derivative of \(\arcsin x\)

Let \(y = \arcsin x\), then \(x = \sin y\). Differentiating implicitly:

\[\frac{dy}{dx} = \frac{1}{\cos y} = \frac{1}{\cos(\arcsin x)} = \frac{1}{\sqrt{1 - x^2}}\]

Thus,

\[frac{d}{dx}(\arcsin x) = \frac{1}{\sqrt{1 - x^2}}\]

Derivative of \(\arccos x\)

Let \(y = \arccos x\), so \(x = \cos y\). Differentiating implicitly:

\[\frac{dy}{dx} = -\frac{1}{\sin y} = -\frac{1}{\sqrt{1 - x^2}}\]
\[\frac{d}{dx}(\arccos x) = -\frac{1}{\sqrt{1 - x^2}}\]

Derivative of \(\arctan x\)

Let \(y = \arctan x\), so \(x = \tan y\). Then:

\[\frac{dy}{dx} = \frac{1}{\sec^2 y} = \frac{1}{1 + x^2}\]
\[\frac{d}{dx}(\arctan x) = \frac{1}{1 + x^2}\]

Summary of All Inverse Trig Derivatives

$$\begin{aligned} &\frac{d}{dx}(\sin^{-1} u) = \frac{1}{\sqrt{1 - u^2}} \cdot \frac{du}{dx},\quad &&\frac{d}{dx}(\cos^{-1} u) = -\frac{1}{\sqrt{1 - u^2}} \cdot \frac{du}{dx} \\ &\frac{d}{dx}(\tan^{-1} u) = \frac{1}{1 + u^2} \cdot \frac{du}{dx},\quad &&\frac{d}{dx}(\cot^{-1} u) = -\frac{1}{1 + u^2} \cdot \frac{du}{dx} \\ &\frac{d}{dx}(\sec^{-1} u) = \frac{1}{|u|\sqrt{u^2 - 1}} \cdot \frac{du}{dx},\quad &&\frac{d}{dx}(\csc^{-1} u) = -\frac{1}{|u|\sqrt{u^2 - 1}} \cdot \frac{du}{dx} \end{aligned} $$

Examples: Derivatives of Inverse Trig Functions

Example 1: Differentiate \(y = \sin^{-1}(x^3)\)

Let \(u = x^3\), so that \(\frac{du}{dx} = 3x^2\)

\[ \frac{dy}{dx} = \frac{1}{\sqrt{1 - u^2}} \cdot \frac{du}{dx} = \frac{3x^2}{\sqrt{1 - x^6}} \]
\[\frac{d}{dx}(\sin^{-1}(x^3)) = \frac{3x^2}{\sqrt{1 - x^6}}\]

Example 2: Differentiate \(y = \sec^{-1}(e^x)\)

Let \(u = e^x\), so that \(\frac{du}{dx} = e^x\)

\[ \frac{d}{dx}(\sec^{-1}(e^x)) = \frac{1}{|e^x|\sqrt{(e^x)^2 - 1}} \cdot e^x = \frac{1}{\sqrt{e^{2x} - 1}} \]
\[\frac{d}{dx}(\sec^{-1}(e^x)) = \frac{1}{\sqrt{e^{2x} - 1}}\]

Example 3: Differentiate \(y = x^2(\sin^{-1}x)^3\)

Let \(u = \sin^{-1}x\), so that \(\frac{du}{dx} = \frac{1}{\sqrt{1 - x^2}}\)

\[ \frac{dy}{dx} = 2x u^3 + 3x^2 u^2 \cdot \frac{du}{dx} \]
\[ \frac{dy}{dx} = 2x(\sin^{-1}x)^3 + 3x^2(\sin^{-1}x)^2 \cdot \frac{1}{\sqrt{1 - x^2}} \]

Integrals Involving Inverse Trig Functions

We can also use the derivatives of inverse trig functions to find their integrals. The following integrals are derived from the derivatives of inverse trig functions:

\[ \begin{aligned} &\int \frac{1}{\sqrt{1 - u^2}} \, du = \sin^{-1} u + C \\ &\int \frac{1}{1 + u^2} \, du = \tan^{-1} u + C \\ &\int \frac{1}{|u|\sqrt{u^2 - 1}} \, du = \sec^{-1} u + C \end{aligned} \]

Example 4: Evaluate \(\int \frac{e^x}{\sqrt{1 - e^{2x}}} \, dx\)

Solution

We can use the substitution \(u = e^x\) to simplify the integral.

Let \(u = e^x\), then \(du = e^x dx\)

Thus, we can rewrite the integral as:

\[ \int \frac{e^x}{\sqrt{1 - e^{2x}}} dx = \int \frac{1}{\sqrt{1 - u^2}} du = \sin^{-1}(u) + C \]

Substituting \(u\) back gives:

\[\int \frac{e^x}{\sqrt{1 - e^{2x}}} dx = \sin^{-1}(e^x) + C\]

Example 5: Evaluate \(\int \frac{1}{1 + 3x^2} \, dx\)

Let \(u = \sqrt{3}x \Rightarrow dx = \frac{du}{\sqrt{3}}\)

\[ \int \frac{1}{1 + 3x^2} dx = \frac{1}{\sqrt{3}} \int \frac{1}{1 + u^2} du = \frac{1}{\sqrt{3}} \tan^{-1}(u) + C \]
\[\int \frac{1}{1 + 3x^2} dx = \frac{1}{\sqrt{3}} \tan^{-1}(\sqrt{3}x) + C\]

Example 6: Evaluate \(\int \frac{dx}{a^2 + x^2}\)

Use substitution \(u = \frac{x}{a}, \quad dx = a du\)

\[ \int \frac{1}{a^2 + x^2} dx = \frac{1}{a} \int \frac{1}{1 + u^2} du = \frac{1}{a} \tan^{-1} \left(\frac{x}{a} \right) + C \]
\[\int \frac{dx}{a^2 + x^2} = \frac{1}{a} \tan^{-1} \left( \frac{x}{a} \right) + C\]

General Integrals for Constant \(a \neq 1\)

\[ \begin{aligned} &\int \frac{du}{\sqrt{a^2 - u^2}} = \sin^{-1} \left(\frac{u}{a} \right) + C \\ &\int \frac{du}{a^2 + u^2} = \frac{1}{a} \tan^{-1} \left( \frac{u}{a} \right) + C \\ &\int \frac{du}{u \sqrt{u^2 - a^2}} = \frac{1}{a} \sec^{-1} \left| \frac{u}{a} \right| + C \end{aligned} \]

Example 7: Evaluate \(\int \frac{dx}{\sqrt{2 - x^2}}\)

\[\int \frac{dx}{\sqrt{2 - x^2}} = \sin^{-1} \left( \frac{x}{\sqrt{2}} \right) + C\]

Example 8: Evaluate \(\int \frac{dx}{\sqrt{9 - x^2}}\)

Let \(u = \frac{x}{3}, dx = 3du\)

\[ \int \frac{dx}{\sqrt{9 - x^2}} = \int \frac{3 du}{3 \sqrt{1 - u^2}} = \int \frac{du}{\sqrt{1 - u^2}} = \sin^{-1}(u) + C \]
\[\int \frac{dx}{\sqrt{9 - x^2}} = \sin^{-1} \left( \frac{x}{3} \right) + C\]

Example 9: Evaluate \(\int \frac{dy}{y \sqrt{5y^2 - 3}}\)

Let \(u = \frac{\sqrt{5}y}{\sqrt{3}}\)

\[ \int \frac{dy}{y \sqrt{5y^2 - 3}} = \frac{1}{\sqrt{3}} \int \frac{du}{u \sqrt{u^2 - 1}} = \frac{1}{\sqrt{3}} \sec^{-1} \left| \frac{\sqrt{5}y}{\sqrt{3}} \right| + C \]
\[\int \frac{dy}{y \sqrt{5y^2 - 3}} = \frac{1}{\sqrt{3}} \sec^{-1} \left| \frac{\sqrt{5}y}{\sqrt{3}} \right| + C\]
